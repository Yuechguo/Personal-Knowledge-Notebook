### AMD GPU CPX mode
![](./Pasted%20image%2020240729105835.png)
With CPX mode, each chiplet is one individual GPU(one XCD equal to one GPU). So one socket equals to 8 GPU.

In AMD SMI API:
``` c
enum amdsmi_compute_partition_type_t {
  COMPUTE_PARTITION_INVALID = 0 , 
  COMPUTE_PARTITION_CPX , 
  COMPUTE_PARTITION_SPX , 
  COMPUTE_PARTITION_DPX ,
  COMPUTE_PARTITION_TPX , 
  COMPUTE_PARTITION_QPX
}
```

![](./Pasted%20image%2020240729110108.png)

How to set CPX mode:
```
rocm-smi --setcomputepartition cpx
```
